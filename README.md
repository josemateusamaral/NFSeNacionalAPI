Este projeto se trata de uma biblioteca básica para emissão da NFSe Nacional usando Java.<br><br>
Até o momento a biblioteca permite a emissão, cancelamento e obtenção dos parametros do município, porem recebera novas operações posteriormente.<br>
Toda a logica das operações fica dentro da classe <b>NFSeNacionalAPI.java</b>.<br>
Para utlizar a biblioteca é possuir certificado e possuir um Truststore (.jks) que contenha os certificados dos endpoints utilizados pela API. Os endpoints podem ser vistos na classe <b>NFSeNacionalAPI.java</b>.
